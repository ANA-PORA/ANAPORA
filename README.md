# ANAPORA

Plataforma web full stack focada no e-commerce de artesanatos e biojoias amazônicas. O sistema adota uma arquitetura em camadas e está em desenvolvimento ativo.

## 🛠️ Tecnologias Principais
*   **Backend:** Python (FastAPI), SQLAlchemy, Uvicorn
*   **Frontend:** React.js, CSS3
*   **Infraestrutura:** PostgreSQL, Docker Compose

## 🚀 Funcionalidades
- [ ] Cadastro e Autenticação
- [ ] Catálogo de Biojoias/Artesanatos
- [ ] Carrinho e Checkout
- [ ] Integração com N8N (Notificações)

## 📦 Início Rápido
```bash
git clone https://github.com/ANA-PORA/ANAPORA.git
cd ANAPORA
docker-compose up -d
```
*Instruções detalhadas de execução (back/front) serão adicionadas no progresso do projeto.*
